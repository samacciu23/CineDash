Welcome to the CineDash Server

Use the following urls to retrieve data.

From SQLite internal database

- Go to (rest framework) API root ===> /api/

- Get list of movies ===> /api/movies/

- Get list of profiles ===> /api/profiles/

From The Movie DataBase "TMDB" (external API service)

- Get movie with id = 11 ===> /tmdb/movies/{id}/

- Get popular movies ===> /tmdb/movies/popular/

- Get top rated movies ===> /tmdb/movies/top_rated/

- Get upcoming movies ===> /tmdb/movies/upcoming/

- Get poster with image size = "original" and with path = "1E5baAaEse26fej7uHcjOgEE2t2.jpg" ===> /tmdb/posters/{size}/{path}/
